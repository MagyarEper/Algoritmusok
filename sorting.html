Keresés, rendezés, mohó stratégia: The Full Counting Sort

Feladat leírása

Adott egy lista, amelyben egy egyész számból és egy sztringből álló listákat tárolunk. A listát rendeznünk kell az egészek szerint, de az első felében lévő sztringeket le kell cserélnünk "-" -jelekre.

Solution Explanation

Először megkerestem a legnagyobb egész számot a listákban, hogy annyi darab (+1) üres listával inicializálni tudjam a megoldási listát. Miközben ezt megtettem a lista felénél kissebb indexű sztringeket kicseréltem "-" -jelekre. Ezután a megoldások listájában lévő üres listákat feltöltöttem azokkal a sztringekkel melyek az eredeti listában az üres lista indexének megfelfelő egésszel voltak párbaállítva. Eztán márcsak az eredmény standart kimenetre való kiírása maradt hátra.

        def countSort(arr):
            # Write your code here
            maximum = int(0)
        
            for i,n in enumerate(arr):
                if i < len(arr)/2:
                    n[1] = '-'
                if int(n[0]) > int(maximum):
                    maximum = int(n[0])
                    
            maximum += 1
            solution = [[] for x in range(0,int(maximum))]
            for m in arr:
                solution[int(m[0])].append(m[1])
            print(' '.join([' '.join(x) for x in solution if x]))
            
                
        if __name__ == '__main__':

                arr.append(input().rstrip().split())
